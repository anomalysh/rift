```console
$ rift http 3000 myapp

  rift  https://myapp.rift.example.com
    ->  http://127.0.0.1:3000
```

One command forwards `http://127.0.0.1:3000` to a public HTTPS URL. A Go gateway
(`riftd`) terminates the agent's WebSocket connection and routes public traffic
for `*.<base-domain>` into it; a Caddy instance in front handles wildcard TLS.
The agent is a single Bun binary. Nothing runs on anyone else's servers.

## What rift is, and is not

rift tunnels **HTTP** and nothing else today. The wire protocol reserves room
for TCP and TLS tunnelling, but this build does not implement them: `http` is
the only accepted protocol. HTTP tunnels, token authentication, reserved
subdomains, application-level heartbeats and automatic cleanup all work and are
covered by tests. Multi-node routing via Redis is implemented but has only ever
been exercised single-node.
